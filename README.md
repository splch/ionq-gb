# IonQ GB

GameBoy game for IonQ Lunch-n-Learn

## Features

1. Graphics
2. Audio
3. Controls
4. Interrupts

## Build

```shell
export GBDK_HOME=/path/to/gbdk/
make gb
```

## Run

- `emulator build/gb/logo.gb`

- Upload [ROM](build/gb/logo.gb) to [GB Emulator](https://wasmboy.app/)
